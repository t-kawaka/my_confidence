# アプリ名：my_confidence

## アプリ概要

自分自身の日々の活動記録を記述することができるサイトです。
自分自身の活動記録を記述することで、「自分はこれだけ頑張った！」ということを実感してもらい
自分自身に裏付けのある自信を持ってもらうためのサイトです。
日々の活動記録をつけたくなるような機能の搭載を目指しています。

## バージョン

* ruby 2.6.3
* rails 5.2.3
